# DataStuctSim

Projeto de Programação Para Dispositivos Moveis, feito em Java no Android Studio.

## Objetivo

Tem por objetivo simular algoritmos de Estrutura de Dados com um pseudocodigo de intuitivo.

## Estrutura

O projeto sera estruturado da seguinte maneira:
| Pasta      | Função                                         |
| ---------- | ---------------------------------------------- |
| **data**   | Busca e entrega os dados (API, local, repos)   |
| **domain** | Lógica de negócio, independente de Android     |
| **ui**     | Telas, ViewModels, código que mexe com Android |
| **di**     | Injeção de dependência                         |
| **utils**  | Classes utilitárias e gerais                   |
